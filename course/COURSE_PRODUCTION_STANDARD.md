# 不锈钢新人学习系统：课程生产金标规范

版本：v1.0（2026-06）

## 1. 产品目标

课程不是资料摘要，也不是把标准拆成视频。每一讲都要让学员完成一个可验证的岗位动作。

每讲必须回答：

1. 学员在什么真实场景中会遇到这个问题？
2. 正确判断应当依据哪一本资料、哪一类条款？
3. 哪些表述是直接依据，哪些是归纳，哪些越界？
4. 学员最终需要交付什么工作样品？

## 2. 每讲固定结构

### A. 情境进入
- 只呈现一个核心冲突。
- 不先公布答案。
- 场景必须来自业务、采购、质检、仓储或加工工作。

### B. 交互探索
- 至少一个真正改变画面或结论的交互。
- 可采用滑块、标签筛选、排序、配对、标准路由、尺寸模拟或材质书标注。
- 不允许仅用“点击下一页”冒充互动。

### C. 证据桌
每个结论显示：
- 来源名称；
- 条款、表格或章节位置；
- 证据性质：直接规定 / 教材解释 / 跨资料归纳；
- 适用范围和边界。

### D. 决策任务
- 必须要求学员作出判断。
- 干扰项应反映行业真实误区，而不是明显笑话。
- 反馈解释“为什么”，不能只显示正确或错误。

### E. 输出任务
至少完成一种：
- 向客户解释；
- 补全询价；
- 形成技术条件；
- 审核材质书；
- 标准定位；
- 判断是否需要补充资料。

### F. 间隔复习
- 24小时：概念辨析；
- 7天：换场景迁移；
- 30天：岗位综合任务。

## 3. 证据纪律

### 绿色：直接依据
资料中有明确条款、表格或原文支持。

### 黄色：跨资料归纳
由两个以上信息点共同支持，但不是标准原句。

### 红色：资料未覆盖或表述越界
必须明确停止，不允许模型凭常识补写。

## 4. 五本资料分工

- GB/T 20878—2024：定义、分类、牌号、统一数字代号、化学成分。
- ASTM A240/A240M—26：美标板、薄板和钢带产品要求。
- GB/T 713.7—2023：承压设备用不锈钢和耐热钢板带的订货、尺寸、性能与检验。
- Handbook of Stainless Steel：组织、元素、腐蚀、生产、加工和选材解释。
- The History of Stainless Steel：历史脉络、商业化、工艺和标准化背景。

资料角色不得互换。

## 5. 视觉与交互规范

- 主背景：象牙白；主色：深酒红；点缀：香槟金与不锈钢银灰。
- 一屏只完成一个主要动作。
- 标题不超过两行，正文主段不超过120字。
- 操作说明放在动作之前，使用一句话说明。
- 下一步按钮在必要交互完成前保持禁用。
- 错误反馈不羞辱学员，只指出判断链缺失的位置。
- 移动端保持单列，操作按钮触控高度不低于44px。

## 6. 课程验收清单

上线前逐项确认：

- [ ] 所有事实均有五本资料依据或明确标记为课程归纳。
- [ ] 标准编号、年份、牌号、化学符号和单位复核无误。
- [ ] 没有把牌号标准写成产品订货标准。
- [ ] 至少包含一个有效交互。
- [ ] 至少包含一个真实判断任务。
- [ ] 至少包含一个自由输出或工作样品。
- [ ] 下一步按钮具有交互门槛。
- [ ] PC与手机布局均通过检查。
- [ ] 键盘可操作，焦点可见，关闭按钮明确。
- [ ] 资料未覆盖的问题不会生成伪答案。

## 7. 课程数据字段

每个知识点至少保留：

```json
{
  "knowledge_id": "K-001",
  "lesson_id": 1,
  "claim": "不锈钢定义中的铬含量至少为10.5%",
  "source": "GB/T 20878—2024",
  "location": "3.1",
  "evidence_type": "direct",
  "scope": "不锈钢定义，不等于具体产品交货要求",
  "review_status": "verified",
  "reviewed_at": "2026-06"
}
```

## 8. 禁止事项

- 不整页复制标准或书籍。
- 不重制完整化学成分表作为付费讲义。
- 不把“行业习惯”冒充标准强制要求。
- 不用昂贵、超级、高端等词替代技术判断。
- 不用选择题数量代替学习质量。
- 不把观看时长当作能力完成度。
